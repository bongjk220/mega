AI 코딩 에이전트

## IDE 통합형 (전용 IDE)
 1. `Codex` :  '두뇌' 모델,  다른 에이전트(예: Cursor, Cline 등)가 가져다 쓰는 엔진 역할
 2. `Windsurf`: Codeium에서 출시한 AI IDE로, 'Cascade'라는 강력한 에이전트 기능을 제공합니다.(*거의 무료*)
 3. `Antigravity` : Cursor나 Windsurf의 강력한 경쟁자로 떠오르는 최신 AI 전용 IDE, 가장 똑똑한 IDE" 후보(*크레딧 겁나 빨리 소모됨*)
 4. `Cursor`: VS Code를 기반으로 제작된 AI 전용 IDE로, 프로젝트 전체를 인덱싱하여 답변합니다.(*50회채팅 무료-크레딧 겁나 빨리 소모됨*)
 5. *PearAI*: 오픈소스를 지향하며, 투명한 AI 코딩 환경을 제공하는 것을 목표로 하는 IDE입니다.
 6. *Zed*: 최근 개발자들 사이에서 가장 핫한 초고속 에디터입니다. 원래 AI 전용은 아니었으나, 최근 'Anthropic'과 협업하여 에이전트 기능을 기본 탑재했습니다. Cursor보다 훨씬 가볍고 빠르다는 평이 많습니다.
 7. *Trae* : (중국)바이트댄스. cursor와 유사(*완전 무료*)
 8. *OpenDevin* (현 All-Hands): 세계 최초의 완전 자율형 AI 엔지니어 'Devin'의 오픈소스 대안입니다. 브라우저에서 실행되지만 로컬 파일과 터미널을 직접 제어하는 하이브리드 형태입니다.
 9. *Swe-agent*: 프린스턴 대학교에서 만든 도구로, GitHub의 이슈(버그)를 주면 AI가 스스로 코드를 고치고 테스트해서 PR(Pull Request)까지 올리는 전문 에이전트입니다.
 10. *MarsCode* : (중국)바이트댄스

## IDE 통합형 (VS Code 확장)
 1. `GitHub Copilot`: 이슈 해결 계획부터 코드 작성까지 전 과정을 에이전트가 관리합니다.
 2. *Cline + OpenRouter* : VS Code 내에서 파일/터미널을 직접 제어(*무료*)
 3. *Cody* : 코드베이스 전체 이해 및 검색에 특화(*500회채팅 무료*)
 4. *Continue* : 특정 모델에 종속되지 않는 오픈소스 AI 코딩 어시스턴트(VS Code / JetBrains 확장) 무료 오픈소스이며, API 키를 연결하거나 로컬 모델을 사용할 때 유용
 5. *Open Interpreter* : 챗GPT의 코드 인터프리터를 내 컴퓨터 터미널로 가져온 도구입니다. "내 바탕화면에 있는 이미지들 전부 PDF로 합쳐줘" 같은 시스템 제어 명령에 가장 특화되어 있습니다.
 6. *Roo Code*: Cline에서 파생(Fork)된 확장 프로그램입니다. 현재 Cline보다 업데이트 속도가 빠르고, 코드를 수정할 때 훨씬 정밀하게(Diff 방식) 작업하여 최근 사용자층이 급격히 이동하고 있는 강력한 에이전트입니다.
 7.  *Void*: Cursor의 오픈소스 대안을 표방하며, VS Code 안에서 독립적인 AI 환경을 구축하려는 프로젝트입니다.
 8. *Devika* : 인도의 개발자가 만든 Devin의 오픈소스 대안으로, 프로젝트 계획 수립부터 코드 작성까지 단계별로 시각화해 보여주는 에이전트입니다.
 9. *Baidu Comate* : 중국 바이두. GitHub Copilot의 대안
 10. *Qwen (Tongyi Lingma)* : 중국 알리바바


 ## IDE 통합형 (기타 확장)
 1. *Junie*: JetBrains의 자율 코딩 에이전트입니다.(*VS 확장 아님*)(IntelliJ, PyCharm, WebStorm에서 작동)

 
## 터미널/CLI 기반 (전문가용)
 1. *Claude Code*: Anthropic의 공식 CLI 도구로, 터미널에서 직접 파일 수정, 테스트, 커밋을 수행합니다.(*유료라서 못써봄*)
 2. *Aider*: 터미널에서 LLM과 짝 프로그래밍을 하며 여러 파일을 동시에 수정하는 인기 오픈소스 도구입니다.
 3. *DeepSeek-V3/R1 CLI* : Aider나 Cline 같은 에이전트에 연결. 압도적 1위
 4. *ModelScope CLI* : 알리바바, 수만 개의 중국산 AI 모델을 로컬로 불러와 실행

## 웹/완전 자율형
 1. `Lovable AI` : (Bolt.new, v0.dev 유사), 대화만으로 전체 풀스택 앱 만들고 디자인 감각이 뛰어나고 백엔드 설정까지 알아서 해줍니다.
 2. `Google AI Studio` : Gemini를 코딩이나 앱에 적용하기 전, 웹에서 직접 테스트하고 튜닝하는 '실험실'(*로컬환경 안됨*)
 3. `Manus (Monica 제공)`: '세상에서 가장 똑똑한 범용 에이전트'를 표방합니다. 코딩은 물론 시장 조사, 데이터 분석, 복잡한 프로젝트 기획 등 사람이 웹 브라우저에서 하는 행동을 완벽히 모방하여 수행합니다.
 4. *OpenAI Operator*: Manus처럼 웹 브라우저를 직접 제어하며 작업을 수행하는 오픈소스 에이전트입니다.
 5. *OpenManus*: Manus의 오픈소스 버전으로, 로컬 터미널에서 실행하여 내 컴퓨터의 자원을 활용하는 자율형 에이전트 프로젝트입니다.
 6. *OpenOperator*: Manus처럼 웹 브라우저를 직접 제어하며 작업을 수행하는 오픈소스 에이전트입니다.
 7. *MindStudio* : 웹 브라우저에서 AI 앱이나 자동화 워크플로우를 설계하는 SaaS형 플랫폼입니다.(Claude Code의 기능을 활용)
 8. *Devin*: 스스로 브라우징, 디버깅을 하며 엔드투엔드 개발을 수행하는 '최초의 AI 소프트웨어 엔지니어'로 불립니다.
 9. *Replit Agent*: 웹 브라우저에서 아이디어만 말하면 배포까지 완료해 주는 에이전트입니다. 
 10. *Bolt.new*: 웹브라우저에서 바로 React나 Next.js 프로젝트를 생성하고 코딩해 주는 에이전트입니다. 초기 프로젝트 구축 단계에서 몇 번의 대화는 무료로 가능합니다.
 11. *v0.dev*: Vercel에서 만든 도구로, 채팅만으로 웹사이트의 UI(프론트엔드) 코드를 생성(Next.js와 Tailwind CSS 환경에 최적화)
 12. *Pynecone (현 Reflex)*: 순수 파이썬(Python)만으로 풀스택 웹 앱을 만들어주는 에이전트형 프레임워크입니다. 코딩 에이전트들이 이 형식을 선호하여 자주 사용합니다.
 13. *Zhipu AI*: 칭화대
 14. *Kimi.ai* : 엄청나게 긴 소스 코드 파일들을 한 번에 읽고 분석
 15. *StepFun* : 수조 개의 파라미터를 가진 모델로 웹 앱 개발 및 논리 추론
